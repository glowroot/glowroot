Comma-separated list of sensitive system properties to mask, e.g. passwords. The wildcard '*' is supported anywhere in the property name.
Comma-separated list of sensitive mbean attributes to mask, e.g. passwords. The wildcard '*' is supported anywhere in the attribute name. Matching is performed against ObjectName:AttributeName, so masking just "password" will not match any MBean attributes, while masking "*:password" will match MBean attributes of exactly password for all MBean object names.
